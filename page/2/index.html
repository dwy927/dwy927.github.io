我主要参考了以下博客，click here 完成了快速搭建

Hadoop+eclipse的开发环境搭建

下载插件

下载地址

插件安装

把hadoop-eclipse-plugin-2.6.0.jar这个插件下载下来

放到以下目录下/Applications/Eclipse.app/Contents/Eclipse/plugins

以上目录有可能不一样，只要找到eclipse的安装目录的plugins目录即可

然后重新打开eclipse即可

遇到的一个小坑坑

我重启eclipse发现什么都没有变化，好像eclipse插件没有起到任何作用。

一直以为会不会是hadoop版本和插件版本不一致所造成的

最后发现：是因为之前启动过eclipse，然后下次启动好像会从历史记录中加载，而不检查有没有新的的插件被安装。

解决：
把存历史记录的那个目录删掉。启动的时候找不到历史记录，就会全部重新加载拉～～～～删掉没有关系啦，启动会自动生成的

然而默认的theme好像有一丢丢丑捏，可以把它换掉咩？当然可以啦(*≧ω≦)

hexo主题选择

百度一下可以搜到很多好看的主题呢，挑自己喜欢的style就好

懒人党可以click here

nexT主题安装

非常简单，按照git里面的步骤，照着敲就好啦！别的主题的步骤也都是这样的。

这边给一个比较详细的文档，宝宝就是照着这个敲的click here

Hexo 安装主题的方式非常简单，只需要将主题文件拷贝至站点目录的 themes 目录下， 然后修改下配置文件即可。具体到 NexT 来说，安装步骤如下。

下载主题

之后的更新可以通过 git pull 来快速更新

启用主题

与所有 Hexo 主题启用的模式一样。 当下载完成后，打开your-hexo-site目录下面的_config.yml文件， 找到 theme 字段，并将其值更改为 next

验证主题

用浏览器打开localhost:4000，就可以看到新主题啦 (⁎⁍̴̛ᴗ⁍̴̛⁎)

修改主题的一些设定

按自己的需求改，怎么美怎么改，前面的文档非常详细。

配置文件的路径 your-hexo-site/theme/next/_config.yml

宝宝就列一下自己改的：

设置Scheme

有三种scheme可以挑

  • Muse - 默认 Scheme，这是 NexT 最初的版本，黑白主调，大量留白
  • Mist - Muse 的紧凑版本，整洁有序的单栏外观
  • Pisces - 双栏 Scheme，小家碧玉似的清新

宝宝用的是Pisces，觉得这个更舒服一点

设置头像

  • 找一张美美的照片存下来
  • 保存到your-hexo-site/theme/next/source/images文件夹下
  • 修改头像文件名为avatar.png （不改也可以，配置文件中名字能对起来就好）
  • 修改之前的主题的配置文件your-hexo-site/theme/next/_config.yml

设置manu

menu就是侧边的菜单栏，可以按照自己想要的进行设置

  • home: 首页
  • archives: 归档
  • categories: 分类
  • tags: 标签
  • about: 关于
  • search: 搜索
  • commonweal: 公益404
  • something: 有料

宝宝设置了 home archives tags categories 这四项

具体设置也是配置那个主题配置文件中的menu选项

home archives 设置完了非常的完美，没有什么问题

tags 和 categories出现404

tags 和 categories可能会出现404，要稍微调整一下下 参考这边可以完美解决

写博客果然好考验耐心，宝宝就不粘贴上面文档的解决方案了，大家自己点进去哦 (*≧ω≦)
