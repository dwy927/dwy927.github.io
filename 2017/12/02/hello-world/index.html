安装Xcode

Hexo的编译可能依赖Xcode，从app store 直接下就好

安装node.js

检验

需要注意的是，Hexo 3.1.1测试的最低版本为0.12，所以安装的版本不要太旧

注册github账户

  • 去git官网注册git账号
  • 建立与你用户名对应的仓库，仓库名必须为your_user_name.github.io
  • 配置SSH Keys

就配置好啦 (⁎⁍̴̛ᴗ⁍̴̛⁎)

安装git部署插件

配置 _config.yml 文件

在Hexo安装的目录中找到 _config.yml 文件，打开文件最后，做如下修改并保存

deploy: 
      type: git 
      repo: https://github.com/用户名/用户名.github.io.git

这样就配置好了

git发布的页面在哪里

https://用户名.github.io/

在这个网址下就是我们搭建好的博客的主页面啦
